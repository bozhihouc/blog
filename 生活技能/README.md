### msfvenom

> 查看编辑器 `msfvenom --list encoders`
>
> 查看 `windows/meterpreter/reverse_tcp`支持的平台 `msfvenom -p windows/meterpreter/reverse_tcp --list-options`
>
> 查看所有payload `msfvenom --list payloads`
>
>还有 ` nops, platforms, archs, encrypt, formats`
