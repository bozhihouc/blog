# docker _ubuntu 容器安装web服务

### 搭建web服务
```angular2html
nginx
php
mysql
```
### ubuntu 安装nginx mysql php
```angular2html
apt-get update
apt-get install apache2 php php-gd php-fpm libapache2-mod-php php-mysql mysql-server

```

### ubuntu 安装网络分析工具包（ifconfig netstat 等）
```angular2html
apt-get install net-tools

```
### ubuntu 查看所有服务
```angular2html
service --status-all
```
